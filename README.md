# Pillexis Labs

Company website for **Pillexis Labs Pvt Ltd** — a software product studio building AI-powered tools for founders, creators, and product teams.

## Quick Start

Open `website/index.html` in any browser. No build tools, dependencies, or setup required.

## Architecture

- **Single file** — all HTML, CSS, and JS lives in `website/index.html`
- **No build system** — static site, open directly in browser
- **No dependencies** — everything inline except Google Fonts (DM Serif Display, Inter)
- **Inline SVG icons** — Lucide-style icons embedded directly, no icon library CDN

## Design System

| Token | Value |
|-------|-------|
| Primary | `#e8622a` (orange) |
| Light accent | `#f28c5a` |
| Backgrounds | `#faf6f1` (cream), `#f7e8d5` (warm), `#1a1a1a` (dark) |
| Heading font | DM Serif Display |
| Body font | Inter |
| Breakpoint | 768px (single responsive breakpoint) |
